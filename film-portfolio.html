Exposure 01 — 实习项目

疆海 AI 客服系统

在广州疆海科技独立搭建的AI多语言智能客服系统，服务海外储能产品线。核心架构是一套级联调度系统：FAQ缓存层做第一道拦截，意图分类层（16种意图，Qwen3-8B）做智能分流，之后并行进入RAG轻量链、Agent重链或人工转接三条通路。

评测体系使用LLM-as-Judge框架（Llama 3.3），从忠实度、准确度、相关性、安全性四个维度打分，配合约500条Golden Dataset做持续监测。数据飞轮机制确保系统能从真实对话中持续学习。

系统部署在AWS Bedrock上，使用Claude 3.5 Haiku/Sonnet做生成，BGE-M3做跨语言嵌入，符合GDPR合规要求。

Tech Stack
LangGraph RAG Qwen3-8B Claude 3.5 BGE-M3 AWS Bedrock LLM-as-Judge
Exposure 02 — 个人项目

IdeaShu 灵感书

基于CrewAI搭建的小红书AI内容创作工作台，5节点多智能体流水线。Agent 1使用GLM-4.7通过SiliconFlow做多工具编排（SerperDevTool、JinaReaderTool、YoutubeVideoSearchTool等），Agent 2-5使用Qwen2.5-7B-Instruct控制成本。

引入ChromaDB做语义去重（余弦相似度阈值0.85），增量式风格学习通过异步diff分析实现——每10条记录触发一次System Prompt重构。Prompt Caching进一步优化了token消耗。

核心指标：95%+溯源成功率，34% token成本降低，50%生成时间缩短，修改率从35%降至14%。

Tech Stack
CrewAI GLM-4.7 ChromaDB Qwen2.5-7B SiliconFlow Prompt Cache
Exposure 03 — 个人项目

MoodMix 情绪调饮

一个以东方哲学（五行/阴阳/时辰）为语义桥梁的情绪饮品推荐应用。多智能体流水线将用户情绪解析为6维向量，通过加权余弦相似度映射到8维饮品特征向量，Kappa灵敏度常数实现动态权重调节。

哲学标签系统采用三标签结构（辨证/策略/体感），语录生成逻辑控制在15-28字的因果结构：状态→饮品→行动。前端React实现东方极简美学，部署在Render上。

Tech Stack
React Node.js 五行/阴阳 Cosine Similarity Multi-Agent Render
